HighlightHelper.js Demo

HighlightHelper.js enables highlighting or underlining text in an HTML page. Highlights are drawn via SVG shapes, the Custom Highlight API, or HTML mark elements. Highlights can be drawn across multiple paragraphs or inline style elements—they can even overlap!

Select text, then choose a color and style. Tap away when you’re done. Tap to activate it again for editing. Secondary elements like images and Japanese ruby/furigana are highlighted around.
